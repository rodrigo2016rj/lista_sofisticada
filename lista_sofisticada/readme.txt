Utilize este plugin para criar listas no editor de posts do WordPress.
O bloco Lista Sofisticada permitirá a você criar listas com:
Título da lista;
Itens de texto;
Listas internas;
Links.

Tudo ilimitadamente, pode colocar quantos itens, quantas listas internas e quantos links você quiser.

Para colocar um link, adicione um item de texto à sua lista e acrescente {endereço URL} após o texto.
Exemplo: WordPress{https://wordpress.org} fará o item ser um link para o site do WordPress.
O mesmo serve para os títulos das listas, também é possível transformar o texto do título em link, a forma é a mesma.

Há vários layouts disponíveis para as suas listas, escolha o que melhor se adapte ao visual do seu site.
